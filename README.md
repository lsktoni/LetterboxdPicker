# LetterboxdPicker

A small utility that **randomly selects a movie from your Letterboxd watchlist**; perfect for indecisive movie nights, group movie sessions, or just when *you can’t decide what to watch next*.

---

## Overview

If you’ve ever stared at your Letterboxd watchlist wondering *“What should I watch tonight?”*, this project solves exactly that.  
**LetterboxdPicker** fetches your watchlist and picks a movie at random, giving you a no‑pressure way to finally get through that list.  

---

## Features

- Randomly pick a movie from your Letterboxd watchlist  
- Simple and lightweight utility  
- Easy to run locally using Python  
- Great for personal watchlist decision making  

---

## Installation and Running

1. **Clone the repo:**

   ```bash
   git clone https://github.com/lsktoni/LetterboxdPicker.git
   cd LetterboxdPicker
   ```

2. **Set up dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3. **Running the program:**
    ```bash
    python server.py
    ```
